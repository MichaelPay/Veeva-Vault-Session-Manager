Privacy Policy

Last updated: 2024-12-03

Introduction

The Veeva Vault Session Manager Chrome extension ("the Extension") is designed to help users manage their Veeva Vault sessions by displaying cookie, session, and user information directly within the browser. This Privacy Policy outlines how the Extension handles user data and ensures the protection of your information.

Information Collected

The Extension collects the following information locally within your browser:

  • Cookies: Specifically, the Extension accesses cookies named JSESSIONID, TK, and veevaUserSsoSettings from domains related to Veeva Vault (e.g., *.veevavault.com).
  • Session Storage Data: User session information stored within the browser's session storage for Veeva Vault domains.
  • User Information: User ID and username extracted from session data to display within the Extension interface.

Use of Information

The information collected by the Extension is used solely for the purpose of displaying session and user details to you within the Extension interface. This allows you to manage your Veeva Vault sessions more effectively.

Data Transmission

No data collected by the Extension is transmitted to any external servers or third parties. All data processing and display occur locally within your browser.

Data Storage and Retention

The Extension does not store or retain any personal data beyond the current browser session. All information is accessed in real-time and displayed directly to you without permanent storage.

Permissions

The Extension requires certain permissions to function correctly:

  • Cookies Permission: To access and read specific cookies associated with your Veeva Vault sessions.
  • Tabs Permission: To query open tabs for Veeva Vault domains to retrieve session storage data.
  • Host Permissions: Access to https://*.veevavault.com/* to read session storage and interact with Veeva Vault pages.

These permissions are necessary for the core functionality of the Extension and are not used for any other purposes.

Security

The Extension prioritizes the security of your data by:

  • Processing all data locally within your browser.
  • Not transmitting any data over the internet.
  • Using standard Chrome extension security practices to prevent unauthorized access.

Third-Party Services

The Extension does not use or integrate with any third-party services that could access your data.
